## Building
Folow the instructions of the [VCV Rack manual](https://vcvrack.com/manual/Building.html).
Switch to the `plugins/` directory then:

  ```
  git clone https://github.com/AnimatedCircuits/RackModules.git
  cd RackModules
  make
  ```
